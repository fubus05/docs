## Don't see your platform?

If your affiliate program is not listed, use the [generic REST integration](/partner-integration/platforms/generic-rest). You'll need to configure date parameters and potentially map the response fields yourself.

If the API response format doesn't match the generic parser, contact support with a sample API response and we can add a dedicated parser.
